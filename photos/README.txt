Tips
----
• .jpg, .png, .webp all work — just keep the "photoN" name.
• If you use .png/.webp instead of .jpg, tell me and I'll update the manifest.
• Order matters: photo1 shows first, photo25 last.
• You can pair each photo with a memory caption in  src/content.js
  (the `memories` list) — photo1 ↔ first memory, and so on.

That's it. Save the files, refresh the page, and they appear. ♥
